# Screenshots do BTC Cycle Monitor

## Como adicionar os prints ao README

1. Salve os prints das telas nas seguintes localizações:
   - `docs/screenshots/dashboard.png` - Tela principal com gráfico e estatísticas
   - `docs/screenshots/alerts.png` - Tela de alertas e preferências
   - `docs/screenshots/indicators.png` - Tela de indicadores técnicos

2. Certifique-se de que as imagens estão em formato PNG ou JPG

3. Tamanho recomendado: 1920x1080 ou 1366x768

## Prints necessários

### 1. Dashboard Principal (dashboard.png)
- Captura da tela mostrando:
  - Gráfico do Bitcoin
  - Preço atual e variação
  - Estatísticas 24h (máxima, mínima, volume, etc)
  - Botões de período (1D, 1W, 1M, etc)

### 2. Alertas e Preferências (alerts.png)
- Captura da aba "Alertas" mostrando:
  - Campo de alerta de preço
  - Campo de alerta de oscilação
  - Toggle switches ativos
  - Configurações de moeda, tema, etc

### 3. Indicadores Técnicos (indicators.png)
- Captura da tela de indicadores mostrando:
  - Fear & Greed Index (gauge)
  - Pi Cycle Top Indicator
  - Dominância BTC

## Dicas para boas screenshots

- Use o app em resolução HD (1920x1080)
- Certifique-se de que os dados estão carregados
- Evite informações pessoais ou sensíveis
- Prefira modo escuro para melhor visualização
- Use a ferramenta de captura do Windows (Win + Shift + S)
